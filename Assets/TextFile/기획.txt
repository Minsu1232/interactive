1. 프로젝트 개요
1.1 기본 정보

장르: 2D 클리커 기반 실감형 인터랙티브 체험 콘텐츠

예상 플레이타임: 3-5분

투자 기초 교육을 게임화한 체험형 콘텐츠 제작

4. 기술 사양
4.1 개발 환경
소프트웨어:

Unity 6 (2024 LTS)
Visual Studio 2022
ComfyUI (로컬 서버)
Git (버전 관리)

하드웨어 요구사항:

GPU: GTX 1060 이상 (ComfyUI 구동용)
RAM: 16GB 이상
Storage: 50GB 이상 (AI 모델 저장용)
프린터: A4 컬러 프린터

4.2 Unity 패키지 구성
필수 패키지:
- UI Toolkit (모던 UI 시스템)
- TextMeshPro (텍스트 렌더링)
- Newtonsoft JSON (API 통신)
- Universal Render Pipeline (최적화)
- Addressables (리소스 관리)
4.3 외부 API 연동
데이터 수집 API:

CoinGecko API (암호화폐 데이터, 무료)
Yahoo Finance API (주식 데이터, 비공식)
Alpha Vantage API (보조 주식 데이터)

AI 이미지 생성:

ComfyUI 로컬 서버 (포트 8188)
SDXL Base 모델 + LifeStyle LoRA

4.4 파일 출력 시스템

iTextSharp (PDF 생성)
Windows Print API (프린터 연동)
A4 신문 레이아웃 템플릿


5. UI/UX 설계
5.1 화면 구성
메인 게임 화면:

상단: 현재 턴, 잔여 시간, 총 자산
중앙: 15개 종목 버튼 (3×5 격자)
우측: 포트폴리오 현황 패널
하단: 뉴스 티커, 매수/매도/보유 버튼

종목 버튼 정보:

종목명 + 아이콘
현재 가격
변동률 (색상으로 표시)
섹터별 색상 구분

5.2 인터랙션 디자인
터치/클릭 기반:

종목 선택: 클릭
수량 조절: 슬라이더
확인/취소: 버튼
30초 타이머 + 진행바

5.3 다국어 지원
지원 언어: 한국어, 영어
번역 대상: UI 텍스트, 종목명, 결과 텍스트
구현 방식: Dictionary 기반 런타임 변경

6. 아트 리소스
6.1 ComfyUI 파이프라인
통일된 아트 스타일:

베이스 스타일: "clean flat design, minimalist UI, 2.5D isometric"
컬러 팔레트: 섹터별 구분 (파랑, 초록, 노랑, 주황, 빨강)
해상도: 512×512 (아이콘), 1024×1024 (라이프스타일)

자동 생성 리소스:

종목 아이콘 15개
UI 버튼 요소들
라이프스타일 이미지 4종
배경 및 장식 요소

6.2 리소스 관리
파일 구조:
StreamingAssets/Generated/
├── Icons/ (종목 아이콘 15개)
├── UI/ (버튼, 패널 등)
├── Lifestyle/ (결과 이미지 4종)
└── Background/ (배경 요소)
